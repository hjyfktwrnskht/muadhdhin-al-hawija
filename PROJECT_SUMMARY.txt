📋 خطوات البناء السريعة
─────────────────────────────────────────────────────────────────────────────
1. فتح المشروع في Android Studio
2. انتظر تحميل الاعتماديات
3. انقر على Build > Build APK(s)
4. انتظر انتهاء البناء
5. ملف APK سيكون في: app/build/outputs/apk/debug/app-debug.apk

أو من سطر الأوامر:
  ./gradlew assembleDebug

🚀 التثبيت
─────────────────────────────────────────────────────────────────────────────
من Android Studio:
  - توصيل الجهاز عبر USB
  - انقر على Run > Run 'app'

من سطر الأوامر:
  adb install app/build/outputs/apk/debug/app-debug.apk

🔄 دورة حياة التطبيق
─────────────────────────────────────────────────────────────────────────────
1. عند بدء التطبيق:
   - بدء Foreground Service
   - طلب الأذونات المطلوبة
   - جدولة الأذان للصلاة القادمة

2. عند وقت الصلاة:
   - تشغيل AlarmManager
   - تشغيل الأذان عبر MediaPlayer
   - عرض إشعار بالأذان

3. عند إغلاق التطبيق:
   - الخدمة تستمر في العمل
   - الإشعار يبقى مرئيًا
   - الأذان يستمر في التشغيل في الأوقات المحددة

4. عند إعادة تشغيل الجهاز:
   - BootReceiver يستقبل الإشارة
   - يبدأ Foreground Service تلقائيًا
   - يعيد جدولة الأذان
